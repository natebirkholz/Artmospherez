# Artmospherez

Weather app using public domain art works to display the current and forthcoming weather. V1.0 currently in review with Apple.

Demonstrated skills:
1. JSONSerialization
2. Parsing JSON into models
3. URLSession
4. Grand Central Dispatch
5. Interface Builder
6. Error Handling
7. Animated UIViewController Transitions
8. CoreLocation
9. Activity Indicator
10. UITableView, custom UITableViewCell
11. Gesture Recognizers
12. Subclassing
13. Inheritance
14. UI/UX
15. MVC
16. File system
17. UserDefaults
18. Apple submission
